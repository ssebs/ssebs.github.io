Home

Welcome to ssebs. We at ssebs are striving to make the most mediocre games with the best-ish gameplay. You might think this is a strange way to run a company, but I, Sebastian, am the sole developer at ssebs.

I hand-write all of my code in Java using eclipse IDE. If you'd like to view my source code, you can find it on Github.

Occasionally I'll get some design help from my (basically) brother, Tyler. He is the one to blame for all of the "artwork" in the games. Most of these games were developed during our AP Computer Science class and I basically carried him during these projects.
